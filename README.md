
# 🔢 NovaCalc – Dynamic Web Calculator

A modern and responsive web-based calculator built using HTML, CSS, and JavaScript with live result preview and keyboard support.

---

## 🚀 Features

- ✅ Live calculation preview
- ✅ Single digit delete (⌫)
- ✅ Full clear (AC)
- ✅ Keyboard support
- ✅ Animated aesthetic background
- ✅ Mobile-style button layout
- ✅ Smooth hover & click effects
- ✅ Responsive design

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Grid + Animations)
- JavaScript (DOM Manipulation)
